========================
STEAM CARD MARKETPLACE
========================
This project can be ran two different ways, either on NetBeans with the source code or by using a .Jar file.
1) NetBeans with source code
- Download the GitHub repository
- Open NetBeans and select the SteamTradingCardProject from the directory you downloaded it to.
- Once the project has been opened click the "Run" button or press the F6 key.

2) .Jar file
- Download the GitHub repository
- Locate the zipped "dist" folder and unzip all of it's contents to the same location
- Double-click on "SteamTradingCardProject.jar" to run

WARNING: Must be connected to the internet in order for the application to function properly.

========================
User Instructions
========================
1) After this folder is unzipped, the user can run SteamTradingCardProject.jar by double clicking it.
2) Both drop down menus can be searched through by typing the desired keys sequentially.
3) Once a card set is selected from the drop down menu click the display button to display the card set.
4) In order to use the price filter and/or the number of cards filter, you must check the desired checkbox(es) and then type numerical values into them. For price you can enter as "1.25" or "125" for the value one-dollar and twenty-five cents.
5) After the desired filter(s) have been applied you must click update to populate the second drop down menu. 
6) Then after a game is selected from the second drop down you can click the second display button under the drop down to display the card set.
7) The user may now click on a card that is displayed to be redirected to the Steam page for buying/selling of the card.

==================
Features
==================
- Searchable drop down
- Maximize, minimize, restore, and resize window
- Double click top drag bar to maximize/restore
- Filters
- Hyperlink for each card displayed

====================
Known Bugs
====================
- Double click to maximize does not work on Windows 10
- The drop down menus have slight white edges when displayed on OSX
- Sometimes you have to click "Display" twice in order for it to register the click on OSX
